Welcome to Text Summarizer

In a recent advance, the significance of text summarization accomplishes more attention due to data inundation on the web. About the project

Summarize

Providing the best crux or summarization for the documents available in various extensions. The summarized data will be sent on the mail and the user will get notified. NLTK library which is used helps us to work with text data.

The user can extract summary by two methods i.e. using Feature Extraction (using Classical NLP methods) and BERT Summarizer (using Deep Learning Concepts).

Question Answering

Answering the user queries with top extracted key fragments and keywords specific to the question for the documents available in all the extensions.

We have used ALLENNLP library which makes it easier to design and evaluate deep learning models and also answers the most relevant information to the user's queries.

Analyse

Summary Evaluation is a very important aspect. We can use various extrinsic (task-based performance measure) and intrinsic(human evaluation) measures to analyse the quality of our summary.

Here, we have trained and tested our text data using Naive Bayes Theorem and further calculated the accuracy of our retrieved summary.
